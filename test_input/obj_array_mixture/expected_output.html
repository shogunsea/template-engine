Arrays are ordered, integer-indexed collections of any object. YOLO BC1 YOLO BC2 Array indexing starts at 0, as in C or Java. A negative index is assumed to be relative to the end of the array---that is, an index of -1 indicates the last element of the array, -2 is the next to last element in the array, and so on.
